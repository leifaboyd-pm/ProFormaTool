Contribution-Based Revenue (CBR)
Any LO can close a CBR deal at any time. A CBR deal is any loan priced below full margin — the gap is rate relief to the borrower. On CBR, corp takes a percentage (not the flat LP rate), and the branch gets the rest.
At defaults: 200 bps × 65% = 130 bps to branch per CBR deal (corp gets 70 bps; LO comp paid from the 130).

Step 1 of 2 — One-Time Setup

Connect your Firebase vault.

The Library stores your prospects in your own Firebase project. Paste two values from your Firebase console — that's it.

Don't have a project yet? Go to console.firebase.google.com → create project → Build → Firestore Database → Create (start in test mode) → Project Settings → Web app → copy the apiKey and projectId.

Then in Firestore Rules, paste:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /vaults/{vaultId}/{document=**} { allow read, write: if true; } } }

The PIN you set next will namespace your data — your vault is not discoverable without it.

Step 2 of 2 — Set Your Vault PIN

Set the key to your vault.

Your PIN namespaces your data in Firestore. Pick something memorable — 4 or more characters, case-sensitive, any combination of words, numbers, or symbols. Do not use a password you use anywhere else.

Important: Your PIN is the key to your vault. If you lose it, your stored prospects cannot be recovered through this tool. Pick something you'll remember.
